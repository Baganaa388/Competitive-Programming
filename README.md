### About

This repository contains my solutions to competitive programming problems from various online judges. Most solutions are written in **C++17** focusing on clean, readable code and learning new algorithmic techniques.

### Compile & run

```bash
g++ -O2 -std=c++17 solution.cpp -o solution
./solution < input.txt
```

Debug mode:

```bash
g++ -O0 -g -std=c++17 -fsanitize=address,undefined -DDEBUG solution.cpp -o solution
```
